# Bonsai Homepage Variant Prompts

These prompts generated the hidden design-board images `D06` through `D09`.

## D06 - 首页方向 A：理念 + 优选项目

```text
Use case: ui-mockup
Asset type: desktop web app screenshot, 1440 x 1024, homepage concept for 盆景 Bonsai
Primary request: Create an alternate homepage concept that explains the site idea first, then offers clear project entry points through curated projects and recent updates.
Scene/backdrop: A polished web app homepage for an independent developer's project garden, not a marketing landing page.
Subject: Bonsai homepage concept A, manifesto plus curated shelf.
Style/medium: realistic production UI screenshot, refined minimal editorial layout, warm white app interface, no browser chrome.
Composition/framing: top navigation with compact brand, first viewport has a strong concise idea statement on the left and a curated featured project shelf on the right; below it, a split area with "优选项目" cards and "最近更新" activity list. Keep a hint of lower content visible.
Lighting/mood: calm daylight, precise, thoughtful, personal, high trust.
Color palette: warm white, ink black, moss green, muted terracotta, pale amber, one restrained sky-blue accent.
Typography: crisp modern CJK UI typography, compact headings, readable Chinese labels.
Text (verbatim): include only short readable labels: "盆景 Bonsai", "一个独立开发者的项目盆景园", "把想法、原型和线上小工具持续养起来", "优选项目", "最近更新", "Factory Soon", "Prompt Shelf", "Tiny Metrics", "正在运行", "周末原型", "查看项目".
Constraints: UI must feel like the actual first screen of a usable site; show project entrances clearly; cards and controls use 8px radius or less; no nested cards; text must not overlap; make all text legible and inside containers.
Avoid: no oversized SaaS hero, no purple gradients, no decorative orbs, no fantasy garden illustration, no cartoon, no fake unreadable text, no dark slate dashboard, no browser chrome, no marketing stock imagery.
```
